# Jasaku - Aplikasi Flutter

Aplikasi mobile untuk layanan jasa.

## API Backend
Backend API untuk aplikasi ini tersedia di repo terpisah: [jasaku_api](https://github.com/RafifFarandHariri/jasaku_api)

## Struktur Proyek
- **Frontend**: Repo ini (Flutter)
- **Backend**: [jasaku_api](https://github.com/RafifFarandHariri/jasaku_api)

## Setup
1. Clone repo ini
2. Clone juga repo API: `git clone https://github.com/RafifFarandHariri/jasaku_api`
3. Jalankan API terlebih dahulu
4. Jalankan aplikasi Flutter
